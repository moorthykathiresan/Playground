Steps to run:

1) Clone the Repo
2) Open Command prompt (or any of your fav shell)
2) Navigate to the directory where  "package.json" exists.
3) run "npm i" or (npm install)
4) run "npm start"

Dependency: Node js needs to be installed.

https://nodejs.org/en/
